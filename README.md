# anydoor

npm i -g anydoor

## 使用方法

```
anydoor # 把当前文件夹作为静态资源服务器根目录

anydoor -p 8080 # 设置端口为8080

anydoor -p localhost # host 为 localhost

anydoor -d /usr # 设置跟目录为usr
```
